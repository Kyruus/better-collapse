Accordion bootstrap-accordion.js

About

Expands upon the bootstrap collapse plugin to create a better accordion element.

* Requires the Bootstrap Transitions, and Collapse plugins to be included.

Usage

  1. Create an accordion as you would normally with bootstrap.
  2. Add a data-toggle="accordion" to each .accordion-group element.
  3. Add the class js-close-target to any children of the accordion-group that should be visible only when the group is closed.
  4. Add the class js-open-target to any children of the accordion-group that should be visible only when the group is open.

Options

  1. Adding the attribute data-open-target=".icon-minus" to an accordion group will change the default closed target class to icon-minus as opposed to js-close-target
  2. The attribute data-close-target=".icon-plus" performs the same override to the open class.
