Better Collapse better-collapse.js

About

Expands upon the Bootstrap collapse plugin to dynamically hide and show elements inside the toggler.

* Requires the Bootstrap Transitions and Collapse plugins to be included.

Usage

  1. Create a collapse element as you would normally with Bootstrap.
  2. Add a data-hide-on="collapsed" to any children of the data-toggle="collapse" element that should be hidden when the toggled element is collapsed.
  3. Add a data-hide-on="expanded" to any children of the data-toggle="collapse" element that should be hidden when the toggled element is expanded.

      <span data-hide-on="collapsed">Hide</span>
      <span data-hide-on="expanded">Show</span>
    </a>
  </p>
  <div id="toggled-section" class="collapse">
    Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod

</div>
